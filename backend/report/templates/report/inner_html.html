This report summarizes your reflection about the role of GenAI in the above mentioned course. The aim of this report is to stimulate reflection on the impact of generative AI on academic teaching. Students can use generative AI for almost all assignments. The question whether assessment forms need to be adapted as a consequence of this, depends on the intended learning outcomes that the assignments test. This is why this report aims to make an explicit link between assessment forms and learning outcomes. Wherever the use of generative AI is considered problematic, the tool provides alternative assessment forms, or stimulates you to consider changing the learning outcomes.
